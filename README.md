# Proyecto PIA

**Objetivo general:** Desarrollar un payload académico y benigno que ejecute comandos limitados para auditorías de seguridad en entornos controlados, garantizando ausencia de persistencia.

**Plataforma objetivo:** Linux con ejecución en máquinas virtuales aisladas de laboratorio.

## Instrucciones para clonar el repositorio

```bash
git clone https://github.com/DavidRdzG/PIA-PAC
```
